Our Project basically focuses not on the COVID-19 problem rather it will provide the solution for the next step. As we have witnessed 2 massive waves of COVID-19 and a semi 3rd Wave.

Powerful

We have built a highly powerful and efficient tool which can run on any machine.It is very light weight too which means you can use this tool in low bandwidth connection as well.

Gestures Reader

Our team designed a highly trained and advanced tool powered by advanced machine learning algorithms which can detect and read customers hand gestures which will indicate whether they are vaccinated or not with an accuracy of 95%.

Advanced Graph

Our team designed a very advanced graph analysis tool which can analyse the current statistics of the world in real time.You do not need to worry about about outdated data because out data constanly updates in real time Covid counts.

In-house navigation

We designed a in-house advanced mapping system which will fetch data from all over the world and provide you the details of the locations in real time.Our map also feature some advanced filters which will indicate the vaccine status near you.
